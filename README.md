# Rayavriti Website

A professional website for Rayavriti - Technology & Networking Solutions.

## Directory Structure

```
├── assets/images/    # Logos and icons (SVG)
├── css/              # Stylesheets
├── docs/             # Design documentation
├── index.html        # Home page
├── services.html     # Services page
├── contact.html      # Contact page
├── 404.html          # Error page
├── favicon.svg       # Site icon
├── robots.txt        # SEO rules
└── sitemap.xml       # SEO sitemap
```

## Development

Open `index.html` in a browser to preview the site.

## Deployment

This is a static website. Deploy to any static hosting:
- GitHub Pages
- Netlify
- Vercel
- Cloudflare Pages

Simply upload all files (excluding `/docs/` folder if desired).
